# 식품 알레르기 환자를 위한 애플리케이션

### 📱우리의 앱은

- **`*알레르기*`로 마음대로 먹지도 못하는 당신을 위한 맞춤형 관리 앱** <br>
- **`*필터링*` 서비스로 개인이 원하는 정보만 빠르게 확인** <br>
- **`*ocr,바코드*` 기술로 간편하게 찾아보는 음식 성분들!** <br>
- **`*AI*`에게 추천받는 음식 레시피와 식당!**

<hr>

### ⏲️ 개발 기간

- 2023.09 ~ 2024.10
